# Nexora Smart Attendance AI

Smarter Presence Management

## Overview

Nexora Smart Attendance AI is an AI-powered attendance management system developed using FlutterFlow, Firebase, and Face++ API.

## Features

- User Authentication
- Student Enrollment
- Face Registration
- Attendance Tracking
- Teacher Dashboard

## Technology Stack

- FlutterFlow
- Firebase Authentication
- Cloud Firestore
- Face++ API
